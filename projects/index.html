Read n' Ride

View on GitHub

Read n' Ride is a web app that finds articles based on your estimated Metro commute time. You input your starting and destination stations, and receive up to five random pieces of writing in return. The app is built with Meteor and uses the WMATA API and the following API, which I built.

Longreads scraper + API

longreads-api.herokuapp.com View on GitHub

To learn Express, I built a small API to serve data scraped from Longreads.com, a fantastic long-form journalism aggregation site. The scraper relies on the Node cheerio module and data is persisted to a MongoDB instance.
